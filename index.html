Info

Tältä sivulta voit seurata kolmen suositun krypton hintoja. Vihreästä napista painamalla voit päivittää hinnat. Kun nappia painetaan, se muuttuu punaiseksi ja menee "jäähylle" 60 sekuniksi. Tämä cooldown tapahtuu siksi, koska käytän Coingeckon ilmaista APIa, jotta saan kryptojen hinnat näkymään sivulla. Tämä ilmainen API sallii vain tietyn määrän hinnanpäivityspyyntöjä minuutissa. Mikäli se ylittyy, hintojen tilalle tulee viesti "Virhe". Sivua tehdessäni käytin apuna Chatgpt:tä esim: cooldown-toiminnon tekemiseen, sivun ulkoasun muokkaamiseen ja muuhun sommitteluun.
